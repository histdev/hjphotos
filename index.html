Android 앱 설치

  1. * 앱 설치전 "출처를 알 수 없는 앱 설치" 허용으로 변경 해주세요."
  2. Android: 다운로드
  3. 다운로드 받은 앱을 파일 관리자에서 더블 클릭 -> 
  4. "구글 Protected 보안 경고" 팝업 발생 시 "무시하고 설치" 해 주세요.
  5. - 설치완료 -
  6. 앱 실행 및 로그인 후 사용

회원 가입

  1. "No account yet? Create One" 탭
  2. e-mail과 이름, 비밀번호를 입력 후 등록버튼 탭
  3. 입력한 이메일 계정으로 이동하여 이메일 인증 링크를 클릭하여 준다 (메일 계정 스펨함 확인 필요..)
  4. 링크 클릭 시 인증 화면이 나타나며 이후 로그인 가능
  5. - 가입 완료 -

사진 등록

  1. "+"버튼 탭
  2. 엘범 또는 카메라에서 사진 선택
  3. 해시 태그 입력 후 "공유"
  4. - 등록 완료 -

프로필 사진 변경

  1. 개인화면 단추버튼 탭
  2. "Edit Profile" 메뉴 탭
  3. "Change Photo" 링크 탭
  4. 엘범 또는 카메라에서 사진 선택 후 저장
  5. - 등록 완료 -
